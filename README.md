<h1 align = "CENTER"> AMIGO SECRETO :) </h1>

**DESCRIPCION**

  Este es un proyecto de JavaScript que permite agregar amigos y hacer un sorteo eligiendo a uno de los participantes de manera aleatoria. Permitiendo organizar tus reuniones de manera más fácil y rápida.
  
  **INSTRUCCIONES**
  
  1. Ingresa a la página de amigo secreto.
  2. Escribir el nombre en la casilla de cada integrante.
  3. Hacer Clic en añadir.
  4. Repetir el paso anterior hasta que esten agregados todos los participantes.
  5. Posteriormente, hacer clic en el botón sortear.
  6. Listo, ya tenemos a un ganador. 

  **TECNOLOGIAS UTILIZADAS**

  - HTML
  - CSS
  - JAVASCRIPT
